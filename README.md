# 💄 LizzyJay Cosmetics Branding + OpenCV Project

## 📌 Description
This project combines creative branding design with technical image processing.

The branding part includes:
- Logo Design
- Business Card
- Infographic

The technical part includes:
- Image processing using Python and OpenCV

## 🎯 Objectives
- Build a complete brand identity
- Apply design principles (color, typography)
- Perform image processing operations

## 🛠️ Tools Used
- Adobe Illustrator
- Python
- OpenCV

## 📂 Project Structure
opencv/ → Python code
images/ → original images
screenshots/ → output results

## ▶️ How to Run
Install dependencies:
pip install -r requirements.txt

- Color conversion (RGB, HSV)
- Thresholding
